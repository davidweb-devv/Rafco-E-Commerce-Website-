# Rafco-E-Commerce-Website-
A responsive, mobile-first e-commerce storefront designed for a fashion boutique. Features a dynamic shopping cart system and integrated WhatsApp checkout API to streamline customer-to-seller communication. Tech: HTML5, CSS3 (Custom UI), JavaScript (DOM Manipulation

Key Features:
Mobile-First Design: Optimized for smooth scrolling and easy tapping on handheld devices.
Persistent Cart: Users can add multiple items and see their total before ordering.
Instant Checkout: Leveraging the WhatsApp API for quick, personalized transactions
